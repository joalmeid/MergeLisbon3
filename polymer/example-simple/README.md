polymer
=======

Polymer Meetup - Jan 2014


Setup
=======
npm install
cd public && bower install

Add Your API Keys
=======
cd public/js
edit apikeys.js like so:
```var IMPORT_IO_KEYS = {
    userGuid    : "YOUR_USER_GUID",
    apiKey      : "YOUR_API_KEY",
    sourceGuid  : "DATA_SOURCE_GUID"
};```


Run the Node server
=======
node index.js
